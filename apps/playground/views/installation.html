Installation Guide

Learn how to install and configure Mayvio UI for React, Angular, and Vanilla web projects.

React Installation

Install the React package and its peer dependencies via npm.

npm install @mayvio-ui/react @mayvio-ui/core

Setup

Import the global CSS in your root file (e.g. main.tsx or App.tsx):

import '@mayvio-ui/core/styles.css';

Usage

import { Button } from '@mayvio-ui/react';

function App() {
  return <Button variant="primary">Click Me</Button>;
}

Angular Installation

Install the Angular package via npm.

npm install @mayvio-ui/angular @mayvio-ui/core

Setup

Add the global CSS to your angular.json styles array:

"styles": [
  "node_modules/@mayvio-ui/core/styles.css",
  "src/styles.css"
]

Usage

import { Component } from '@angular/core';
import { MvButton } from '@mayvio-ui/angular';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [MvButton],
  template: `<button mv-button variant="primary">Click Me</button>`
})
export class AppComponent {}

Vanilla Integration

For vanilla HTML/JS projects, install the core package.

npm install @mayvio-ui/core

Alternatively, you can link it directly via unpkg/cdn in your HTML:

<link rel="stylesheet" href="https://unpkg.com/@mayvio-ui/core/styles.css">
<script src="https://unpkg.com/@mayvio-ui/core/scripts.js" defer></script>
